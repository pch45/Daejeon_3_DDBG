# I. 스펙(Specification)
## 구현된 어플리케이션의 주요 기능

- 사용자가 슬랙 채팅을 통해 전송한 텍스트 데이터를 분석한다.

- 텍스트 데이터가 없을 경우 게임 목록을 출력한다.

- 텍스트 데이터가 게임 이름일 경우 게임을 시작한다.

- 게임이 진행중일 경우 텍스트 데이터를 분석하여 게임 결과를 출력한다.

- 유저간 컨텍스트를 분리하여 모든 유저가 각자 게임을 할 수 있도록 한다.
